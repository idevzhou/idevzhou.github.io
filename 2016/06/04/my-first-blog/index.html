博客的第一篇文章，我觉得有必要记录一下这个建站过程。总结踩坑经验，给通过搜索引擎找到这儿的网友一些帮助。

域名

我这个域名 devzhou.com，是在万网上购买的。由于这个域名之前是被别人购买了，所以还去了解了一下域名到期未续费，进入删除期之后多久才能重新购买。国际域名，例如 .com，到期未续费进入删除期后，需要 75 天左右域名才会被彻底删除，然后重新开放注册。

虽然，我的域名是在万网上购买的，但是万网显示到期域名重新注册的时间真是笼统，还没有预约抢注的功能。这一点新网要好很多，新网会明确的告诉你日期，也有预约抢注的功能，价格稍贵几块钱而已。


博客

建站之前查了很多关于 Ghost 搭建博客的资料，但是买最便宜的阿里云 ECS 服务器也要 50+ 每月，考虑到博客才刚刚起步的必要性，斟酌许久就选择用 Hexo 来搭建。

由于 Hexo 是静态博客，所以就没有了服务器配置的一堆事儿，很简单的根据网上教程来就成。

安装 Hexo，遇到权限警告别忘了前面加 sudo。

GitHub or Coding

你可以选择把 Hexo 部署在 GitHub 或者 Coding 上，也可以在两个地方都部署。

一切的步骤都很顺利，但是在绑定域名的时候就出现了问题，在 GitHub 上已经有人建立了一个 CNAME 文件里面写了 devzhou.com，这就导致无法通过这个域名定向到 idevzhou.github.io。

这是 GitHub 给我发的邮件：

The page build completed successfully, but returned the following warning:
The CNAME devzhou.com is already taken. Check out https://help.github.com/articles/troubleshooting-custom-domains/#cname-already-taken for more information.

里面给出的解决方案就是在仓库中删除那个 CNAME 文件，但是那个仓库是别人的，怎么让别人删除呢？好吧，只好出门左转来到 Coding 部署，其实没差，国内访问反而更加方便快捷。


至此，开启博客之旅。
