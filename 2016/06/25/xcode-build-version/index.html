Xcode 自动填写 Version Build 脚本

Posté le 2016-06-25

实际开发过程中，经常要手动改写 Build 版本号，这种无聊的操作很是烦人，运用脚本让 Xcode 自动填写，解放生产力。

Build Path

Xcode Project -> TARGETS -> Gerneral -> Identity -> Build

Script Path

Xcode Project -> TARGETS -> Build Phases -> + -> New Run Script Phase

添加日期

Run Script:

原 Build = 1.0
添加脚本运行后 Build = 1.0.0626


年月日时分秒

Run Script:

原 Build = 1.0
添加脚本运行后 Build = 20160626161220

自增一

Run Script:

原 Build = 1
添加脚本运行后 Build = 2
